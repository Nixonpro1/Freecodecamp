This repository contains Python projects completed as part of the freeCodeCamp curriculum.  
The projects focus on **data analysis, visualization, and statistical computation** using Python libraries such as **Pandas, NumPy, Matplotlib, Seaborn, and SciPy**.

---

## Projects Included

1. **Mean-Variance-Standard Deviation Calculator**
   - File: `mean_var_std.py`
   - Computes the mean, variance, standard deviation, max, min, and sum of a 3x3 matrix.
   - Uses NumPy for calculations.

2. **Demographic Data Analyzer**
   - File: `demographic_data_analyzer.py`
   - Analyzes census data using Pandas.
   - Calculates statistics such as race counts, average age of men, percentage of high earners, and occupation analysis.

3. **Medical Data Visualizer**
   - File: `medical_data_visualizer.py`
   - Visualizes medical examination data using Matplotlib and Seaborn.
   - Includes categorical plots and heatmaps for cardio-related features.

4. **Page View Time Series Visualizer**
   - File: `time_series_visualizer.py`
   - Visualizes forum page views over time with line, bar, and box plots.
   - Shows trends, seasonality, and yearly/monthly averages.

5. **Sea Level Predictor / Visualizer**
   - File: `sea_level_plot.py`
   - Plots historical sea level data with scatter plots and linear regression lines.
   - Predicts sea levels for 2050 and annotates statistical metrics.

## Datasets

All datasets used in these projects are included in the repository:

- `epa-sea-level.csv` – Historical sea level data  
- `medical_examination.csv` – Medical examination data  
- `fcc-forum-pageviews.csv` – FreeCodeCamp forum page views  
- `demographic_data.csv` – Census data
